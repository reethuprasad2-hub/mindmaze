02 • Project Description

Project Description

MindMaze – Escape the AI Lab is an interactive browser-based escape room game developed using HTML5, CSS3, and JavaScript. The game places players inside a futuristic AI research laboratory where a rogue artificial intelligence has taken control of the facility. Players must solve five unique puzzle rooms within a limited time to disable the AI system and escape successfully.

The project was designed to provide an engaging and immersive experience while demonstrating practical frontend web development skills. Each room presents a different logical challenge that tests observation, reasoning, and problem-solving abilities. Features such as a countdown timer, score tracking, hints, progress indicators, local storage, background music, and mission completion screens create a realistic gaming experience.

The target audience includes students, puzzle enthusiasts, and anyone interested in interactive web applications. In the future, the project can be enhanced with multiple difficulty levels, additional puzzle rooms, online leaderboards, user authentication, multiplayer mode, and AI-generated challenges to improve replayability and user engagement.
