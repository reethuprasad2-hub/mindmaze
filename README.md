# 🧩 MindMaze

MindMaze is a browser-based AI-themed escape room game built using HTML, CSS, and JavaScript. Players solve interactive puzzles, race against time, and complete challenges to escape the AI-controlled lab.

## 🚀 Features

- AI-themed escape room gameplay
- Interactive puzzles
- Countdown timer
- Score tracking
- Responsive design
- Background music and sound effects
- Project documentation page

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
